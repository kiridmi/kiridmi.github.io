Smart weather station

Components used:

  • Raspberry Pi 4
  • NodeMCU / ESP8266
  • BME280

Software used:

  • Node-Red
  • Mosquitto
  • Influx DB
  • Grafana

NodeMCU reads the sensor every 2 minutes and broadcasts the data over MQTT. Raspberry Pi reads the sensor also every 2 minutes and also listens to the MQTT broadcasts the collected data is then written to the Influx DB, according to the flow designed in Node-RED. Data can be visualised in real time via the Node-RED dashboard, also historical data can be viewed in Grafana.
